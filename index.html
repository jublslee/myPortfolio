PPAS

Supervised by: Dr. Daniele Schiavazzi & Dr. Alison Marsden

PPAS

• Trained a neural network to achieve hemodynamic simulations with precision exceeding catheter accuracy by 15 times in average for 4 distinct patient models

• Reviewed OOP interface of the 0D lumped parameter network and solver components to ensure yielding targeted pressures and flows when treating multiple stenosis regions in pulmonary arteries while minimizing patient risk

• Incorporated positional encoding in multi-layer perceptrons (MLPs) for geometry-specific neural fields to increase precision up to 47% and executed parallel model solution campaigns on CPUs and GPUs utilizing openMPI, pytorch, and NVIDIA modulus

• Generated low-fidelity 0D and 1D models with single CPUs, vastly reducing computational cost compared to high-fidelity 3D models, which require 50+ CPU cores on high-performance computing Linux cluster

• Tested a multi-fidelity neural network fed on data composed of low-fidelity data (95%) with minimal high-fidelity data (5%) improving runtime by a factor of 50 with only 9% average accuracy loss

LINFA

Supervised by: Dr. Daniele Schiavazzi & Dr. Fang Liu

LINFA

• Supported in developing LINFA; a Python library utilizing normalizing flow and annealing for tractable inference

• Tested with complex models with closed-form solutions and ODE Hemodynamic Models, specifically the Two and Three-element Windkessel Models, on both CPUs and GPUs confirming over 65% decrease in inference cost

• Implemented two GitHub tutorials for 2-dim and 3-dim cases demonstrating parameter inference of a non-identifiable ballistic model using normalizing flows and adaptively trained surrogate models in both PyTorch and TensorFlow

• Publication: Wang Y, Cobian E, Lee J et al., (2024). LINFA: a Python library for variational inference with normalizing flow and annealing. Journal of Open Source Software, 9(96), 6309, https://doi.org/10.21105/joss.06309
